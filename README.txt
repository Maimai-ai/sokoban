	Afin d'éxécuter notre Jeu, Sokoban : 
		- Ouvrir un terminal(invite de commande) dans le dossier ou se trouve le 		README ainsi que le repertoire "src"
		- Puis lancer la commande "ant run"
		- En lançant cette commande, elle génèrera une javadoc, un jar et 		compilera tous nos fichiers .java
	
	Si vous lancer une partie contre un joueur humain, pour faire déplacer le premier 	joueur vous pourrez utiliser les touches Z, Q, S et D (respectivement pour haut, gauche, bas et droite. 
	Et pour le second joueur, il pourra utiliser les flèches du clavier.
